Since the Target Value (MPG) depends on all the other variables, it is a multivariable linear regression problem. Lets start with a single variable linear regression to predict "MPG" from "Horsepower". Since we are using the keras module to train our model, we would first have to start by defining our model architecture. We will use a tf.keras.Sequential model, which represents a sequence of steps. There are two steps we would have to take in our single variable linear regression model:

  • We will have to normalize the Horspower input features using the tf.keras.layers.Normalization preprocessing layer
  • Apply a linear transformation to produce 1 output using a linear layer. For his, we can use the keras Dense layer

def plot_loss(history):
  plt.plot(history.history['loss'], label='loss')
  plt.plot(history.history['val_loss'], label='val_loss')
  plt.ylim([0, 10])
  plt.xlabel('Epoch')
  plt.ylabel('Error [MPG]')
  plt.legend()
  plt.grid(True)
